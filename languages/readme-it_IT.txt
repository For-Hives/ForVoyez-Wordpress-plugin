== Descrizione ==

**Testo Alt Automatico per Immagini** è un potente plugin WordPress che utilizza tecnologia AI all'avanguardia per generare automaticamente testo alternativo di alta qualità ottimizzato per SEO per le tue immagini. Fornendo testo alternativo descrittivo per tutte le tue immagini, questo plugin aiuta a migliorare l'accessibilità del tuo sito web per gli utenti con disabilità visive e a incrementare il posizionamento nei motori di ricerca.

### Caratteristiche Principali

* **Analisi Alimentata da AI**: La nostra AI avanzata analizza le tue immagini per generare testo alternativo accurato e contestualmente rilevante.
* **Ottimizzazione SEO**: Il testo alternativo è ottimizzato per i motori di ricerca per migliorare il posizionamento delle tue immagini nei risultati di ricerca.
* **Conformità all'Accessibilità**: Aiuta a rendere il tuo sito web più accessibile e conforme alle linee guida WCAG.
* **Elaborazione in Batch**: Elabora tutte le immagini esistenti nella tua libreria media con un solo clic.
* **Elaborazione Automatica**: Genera automaticamente testo alternativo per le nuove immagini man mano che vengono caricate.
* **Output Personalizzabile**: Scegli di generare solo testo alternativo o includi anche titoli e didascalie.
* **Supporto Multilingua**: Genera testo alternativo in più lingue per supportare un pubblico internazionale.
* **Interfaccia Intuitiva**: Pannello delle impostazioni semplice e intuitivo per una facile configurazione.

### Per Iniziare

1. Installa e attiva il plugin.
2. Vai a **Impostazioni > Testo Alt Automatico** per configurare il plugin.
3. Inserisci la tua chiave API ForVoyez. (Non ne hai una? [Ottienila qui](https://forvoyez.com))
4. Configura le tue preferenze per lingua, formato di output e altre opzioni.
5. Utilizza l'opzione 'Elaborazione in Batch' per generare testo alternativo per tutte le immagini esistenti.

### Dettagli Tecnici

Il plugin elabora in modo sicuro le tue immagini attraverso l'API ForVoyez per generare testo alternativo accurato e pertinente. Le tue immagini vengono analizzate utilizzando algoritmi avanzati di visione artificiale e elaborazione del linguaggio naturale per creare descrizioni che siano sia informative per gli utenti che ottimizzate per i motori di ricerca.

== Installazione ==

1. Carica i file del plugin nella directory `/wp-content/plugins/auto-alt-text-for-images`, o installa il plugin direttamente dalla schermata plugin di WordPress.
2. Attiva il plugin dalla schermata 'Plugin' in WordPress.
3. Vai a **Impostazioni > Testo Alt Automatico** per configurare le impostazioni del plugin.
4. Inserisci la tua chiave API ForVoyez.
5. Configura le tue preferenze e salva le impostazioni.

== Domande Frequenti ==

= Perché ho bisogno di una chiave API? =

La chiave API è necessaria per accedere ai servizi di analisi delle immagini di ForVoyez che alimentano la generazione del testo alternativo. Puoi ottenere una chiave API registrandoti su [https://forvoyez.com](https://forvoyez.com).

= Posso personalizzare il testo alternativo generato? =

Sì, puoi configurare la lingua, la lunghezza massima e il livello di ottimizzazione per il testo alternativo generato. Inoltre, puoi sempre modificare manualmente il testo alternativo generato dopo che è stato creato.

= Quali formati di immagine sono supportati? =

Il plugin supporta tutti i formati di immagine comuni, inclusi JPEG, PNG, GIF, WebP e altri. Qualsiasi formato di immagine supportato da WordPress può essere elaborato dal plugin.

= Questo plugin rallenterà il mio sito web? =

No, la generazione del testo alternativo avviene in background e non influisce sulle prestazioni front-end del tuo sito web. Quando vengono caricate nuove immagini, il testo alternativo viene generato durante il processo di caricamento senza impattare sull'esperienza utente.

= Come funziona l'elaborazione in batch? =

La funzione di elaborazione in batch scansiona la tua libreria media ed elabora tutte le immagini che non hanno già un testo alternativo. A seconda del numero di immagini, questo processo potrebbe richiedere un po' di tempo per essere completato.

= Questo plugin funziona con siti web multilingua? =

Sì, puoi selezionare la lingua per il testo alternativo generato in modo che corrisponda alla lingua del tuo sito web. Il plugin supporta più lingue tra cui inglese, spagnolo, francese, tedesco, italiano, portoghese, russo, cinese, giapponese e hindi.
