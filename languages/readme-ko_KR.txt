AI 기술을 사용하여 이미지에 대한 SEO 최적화된 대체 텍스트, 제목 및 설명을 자동으로 생성합니다.

== 설명 ==

ForVoyez 이미지 자동 대체 텍스트는 AI 기술을 활용하여 콘텐츠의 이미지에 대해 고품질의 SEO 친화적인 대체 텍스트, 제목 및 설명을 자동으로 생성하는 강력한 WordPress 플러그인입니다. 이 플러그인은 웹사이트의 접근성과 검색 엔진 최적화를 쉽게 개선합니다.

= 주요 기능 =

* 🤖 AI 기반 대체 텍스트 생성
* 🖼️ 기존 이미지 일괄 처리
* 🔄 새로운 업로드에 대한 자동 처리
* 🎨 사용자 정의 가능한 출력 형식
* 🌐 다국어 지원
* 🧰 사용자 친화적인 WordPress 관리 인터페이스

AI 기반 API를 통해 개발자와 콘텐츠 작성자가 SEO를 위해 이미지를 쉽게 최적화할 수 있습니다. 강력한 솔루션으로 시간을 절약하고, SEO를 개선하며, 워크플로우를 간소화하세요.

== 설치 ==

1. 플러그인 파일을 `/wp-content/plugins/auto-alt-text-for-images` 디렉토리에 업로드하거나 WordPress 플러그인 화면을 통해 직접 설치하세요.
2. WordPress의 '플러그인' 화면에서 플러그인을 활성화하세요.
3. 설정 -> ForVoyez 자동 대체 텍스트 화면을 사용하여 플러그인을 구성하세요.
4. 설정 페이지에 ForVoyez API 키를 입력하세요.

= API 키 얻기 =

1. [ForVoyez 가입 페이지](https://forvoyez.com/signup)를 방문하여 계정을 만드세요.
2. 로그인한 후, [ForVoyez 대시보드](https://forvoyez.com/dashboard)로 이동하세요.
3. 대시보드의 API 섹션에서 API 키를 생성하세요.
4. API 키를 복사하여 플러그인 설정 페이지에 붙여넣으세요.

== 자주 묻는 질문 ==

= 이 플러그인을 사용하려면 API 키가 필요한가요? =

네, 이 플러그인을 사용하려면 ForVoyez API 키가 필요합니다. [https://forvoyez.com](https://forvoyez.com)에 등록하여 키를 얻을 수 있습니다.

= 생성된 대체 텍스트를 사용자 정의할 수 있나요? =

플러그인이 자동으로 대체 텍스트를 생성하지만, 필요한 경우 각 이미지에 대해 생성된 텍스트를 수동으로 편집할 수 있습니다.

= 이 플러그인은 모든 이미지 형식과 호환되나요? =

이 플러그인은 JPEG, PNG, GIF, WebP를 포함한 WordPress에서 지원하는 대부분의 일반적인 이미지 형식과 호환됩니다.

= 이 플러그인이 내 웹사이트를 느리게 만들까요? =

아니요, 플러그인은 백그라운드에서 이미지를 처리하므로 웹사이트의 프론트엔드 성능에 영향을 미치지 않습니다.

= 일괄 처리 기능은 어떻게 작동하나요? =

미디어 라이브러리에서 여러 이미지를 선택하고 일괄 작업 드롭다운 메뉴에서 "대체 텍스트 생성" 옵션을 사용하여 한 번에 모두 처리할 수 있습니다.

= 생성된 콘텐츠는 다국어를 지원하나요? =

AI는 여러 언어로 콘텐츠를 생성할 수 있습니다. 플러그인 설정에서 선호하는 언어를 지정할 수 있습니다.

== 스크린샷 ==

1. API 구성 - ForVoyez API 키를 입력하기 위한 간단한 설정 페이지.
2. 이미지 관리 인터페이스 - WordPress 미디어 라이브러리를 관리하고 분석하는 메인 대시보드.
3. 결과 예시 - 다양한 이미지 유형에 대해 자동 생성된 대체 텍스트, 제목 및 설명 showcase.
