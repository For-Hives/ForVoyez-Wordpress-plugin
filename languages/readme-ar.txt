إنشاء نص بديل محسن لمحركات البحث وعناوين وتعليقات توضيحية للصور تلقائيًا باستخدام تقنية الذكاء الاصطناعي. تحسين إمكانية الوصول وتعزيز تصنيفات البحث.

== الوصف ==

**النص البديل التلقائي للصور** هو إضافة WordPress قوية تستخدم تقنية الذكاء الاصطناعي المتطورة لإنشاء نص بديل عالي الجودة ومحسن لمحركات البحث للصور الخاصة بك تلقائيًا. من خلال توفير نص بديل وصفي لجميع صورك، تساعد هذه الإضافة في تحسين إمكانية الوصول إلى موقعك للمستخدمين ضعاف البصر وتعزيز تصنيفات محرك البحث الخاص بك.

### الميزات الرئيسية

* **تحليل مدعوم بالذكاء الاصطناعي**: يقوم الذكاء الاصطناعي المتقدم لدينا بتحليل صورك لإنشاء نص بديل دقيق ومرتبط بالسياق.
* **تحسين محركات البحث**: يتم تحسين النص البديل لمحركات البحث لتحسين ترتيب صورك في نتائج البحث.
* **الامتثال لمعايير الوصول**: يساعد في جعل موقع الويب الخاص بك أكثر إمكانية للوصول ومتوافقًا مع إرشادات WCAG.
* **المعالجة الدفعية**: معالجة جميع الصور الموجودة في مكتبة الوسائط الخاصة بك بنقرة واحدة.
* **المعالجة التلقائية**: إنشاء نص بديل تلقائيًا للصور الجديدة عند تحميلها.
* **مخرجات قابلة للتخصيص**: اختر إنشاء النص البديل فقط أو تضمين العناوين والتعليقات التوضيحية أيضًا.
* **دعم متعدد اللغات**: إنشاء نص بديل بلغات متعددة لدعم الجماهير الدولية.
* **واجهة سهلة الاستخدام**: لوحة إعدادات بسيطة وبديهية لسهولة التكوين.

### البدء

1. قم بتثبيت وتفعيل الإضافة.
2. انتقل إلى **الإعدادات > النص البديل التلقائي** لتكوين الإضافة.
3. أدخل مفتاح API الخاص بـ ForVoyez. (ليس لديك واحدًا؟ [احصل عليه هنا](https://forvoyez.com))
4. قم بتكوين تفضيلاتك للغة وتنسيق الإخراج والخيارات الأخرى.
5. استخدم خيار 'المعالجة الدفعية' لإنشاء نص بديل لجميع الصور الموجودة.

### التفاصيل التقنية

تقوم الإضافة بمعالجة صورك بشكل آمن من خلال واجهة برمجة تطبيقات ForVoyez لإنشاء نص بديل دقيق وذي صلة. يتم تحليل صورك باستخدام خوارزميات رؤية الكمبيوتر المتقدمة ومعالجة اللغة الطبيعية لإنشاء أوصاف مفيدة للمستخدمين ومحسنة لمحركات البحث.

== التثبيت ==

1. قم بتحميل ملفات الإضافة إلى دليل `/wp-content/plugins/auto-alt-text-for-images`، أو قم بتثبيت الإضافة مباشرة من خلال شاشة إضافات WordPress.
2. قم بتفعيل الإضافة من خلال شاشة 'الإضافات' في WordPress.
3. انتقل إلى **الإعدادات > النص البديل التلقائي** لتكوين إعدادات الإضافة.
4. أدخل مفتاح API الخاص بـ ForVoyez.
5. قم بتكوين تفضيلاتك وحفظ الإعدادات.

== الأسئلة الشائعة ==

= لماذا أحتاج إلى مفتاح API؟ =

مفتاح API مطلوب للوصول إلى خدمات تحليل الصور من ForVoyez التي تعمل على توليد النص البديل. يمكنك الحصول على مفتاح API عن طريق التسجيل في [https://forvoyez.com](https://forvoyez.com).

= هل يمكنني تخصيص النص البديل المُنشأ؟ =

نعم، يمكنك تكوين اللغة والحد الأقصى للطول ومستوى التحسين للنص البديل المُنشأ. بالإضافة إلى ذلك، يمكنك دائمًا تحرير النص البديل المُنشأ يدويًا بعد إنشائه.

= ما هي تنسيقات الصور المدعومة؟ =

تدعم الإضافة جميع تنسيقات الصور الشائعة، بما في ذلك JPEG وPNG وGIF وWebP والمزيد. يمكن معالجة أي تنسيق صورة يدعمه WordPress بواسطة الإضافة.

= هل ستبطئ هذه الإضافة موقعي؟ =

لا، يحدث إنشاء النص البديل في الخلفية ولا يؤثر على أداء واجهة موقعك. عندما يتم تحميل صور جديدة، يتم إنشاء النص البديل أثناء عملية التحميل دون التأثير على تجربة المستخدم.

= كيف تعمل المعالجة الدفعية؟ =

تقوم ميزة المعالجة الدفعية بمسح مكتبة الوسائط الخاصة بك ومعالجة جميع الصور التي لا تحتوي بالفعل على نص بديل. اعتمادًا على عدد الصور، قد تستغرق هذه العملية بعض الوقت لإكمالها.

= هل تعمل هذه الإضافة مع مواقع الويب متعددة اللغات؟ =

نعم، يمكنك تحديد اللغة للنص البديل المُنشأ لتتطابق مع لغة موقعك. تدعم الإضافة لغات متعددة بما في ذلك الإنجليزية والإسبانية والفرنسية والألمانية والإيطالية والبرتغالية والروسية والصينية واليابانية والهندية.
