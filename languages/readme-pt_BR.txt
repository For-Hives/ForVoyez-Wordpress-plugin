Gere automaticamente texto alternativo, títulos e legendas otimizados para SEO para suas imagens usando tecnologia de IA. Melhore a acessibilidade e aumente as classificações de pesquisa.

== Descrição ==

**Texto Alt Automático para Imagens** é um poderoso plugin WordPress que usa tecnologia de IA de ponta para gerar automaticamente texto alternativo de alta qualidade e otimizado para SEO para suas imagens. Ao fornecer texto alternativo descritivo para todas as suas imagens, este plugin ajuda a melhorar a acessibilidade do seu site para usuários com deficiência visual e aumentar suas classificações nos motores de busca.

### Principais Características

* **Análise Alimentada por IA**: Nossa IA avançada analisa suas imagens para gerar texto alternativo preciso e contextualmente relevante.
* **Otimização SEO**: O texto alternativo é otimizado para motores de busca para melhorar a classificação de suas imagens nos resultados de pesquisa.
* **Conformidade com Acessibilidade**: Ajuda a tornar seu site mais acessível e em conformidade com as diretrizes WCAG.
* **Processamento em Lote**: Processe todas as imagens existentes em sua biblioteca de mídia com um único clique.
* **Processamento Automático**: Gere automaticamente texto alternativo para novas imagens à medida que são carregadas.
* **Saída Personalizável**: Escolha gerar apenas texto alternativo ou incluir títulos e legendas também.
* **Suporte Multilíngue**: Gere texto alternativo em vários idiomas para atender públicos internacionais.
* **Interface Amigável**: Painel de configurações simples e intuitivo para fácil configuração.

### Começando

1. Instale e ative o plugin.
2. Vá para **Configurações > Texto Alt Automático** para configurar o plugin.
3. Digite sua chave API ForVoyez. (Não tem uma? [Obtenha aqui](https://forvoyez.com))
4. Configure suas preferências de idioma, formato de saída e outras opções.
5. Use a opção 'Processamento em Lote' para gerar texto alternativo para todas as imagens existentes.

### Detalhes Técnicos

O plugin processa com segurança suas imagens através da API ForVoyez para gerar texto alternativo preciso e relevante. Suas imagens são analisadas usando algoritmos avançados de visão computacional e processamento de linguagem natural para criar descrições que são informativas para os usuários e otimizadas para motores de busca.

== Instalação ==

1. Faça upload dos arquivos do plugin para o diretório `/wp-content/plugins/auto-alt-text-for-images`, ou instale o plugin diretamente pela tela de plugins do WordPress.
2. Ative o plugin através da tela 'Plugins' no WordPress.
3. Vá para **Configurações > Texto Alt Automático** para configurar as configurações do plugin.
4. Digite sua chave API ForVoyez.
5. Configure suas preferências e salve as configurações.

== Perguntas Frequentes ==

= Por que preciso de uma chave API? =

A chave API é necessária para acessar os serviços de análise de imagem ForVoyez que alimentam a geração de texto alternativo. Você pode obter uma chave API registrando-se em [https://forvoyez.com](https://forvoyez.com).

= Posso personalizar o texto alternativo gerado? =

Sim, você pode configurar o idioma, comprimento máximo e nível de otimização para o texto alternativo gerado. Além disso, você sempre pode editar manualmente o texto alternativo gerado depois de criado.

= Quais formatos de imagem são suportados? =

O plugin suporta todos os formatos de imagem comuns, incluindo JPEG, PNG, GIF, WebP e mais. Qualquer formato de imagem que o WordPress suporte pode ser processado pelo plugin.

= Este plugin vai deixar meu site mais lento? =

Não, a geração de texto alternativo acontece em segundo plano e não afeta o desempenho front-end do seu site. Quando novas imagens são carregadas, o texto alternativo é gerado durante o processo de upload sem impactar a experiência do usuário.

= Como funciona o processamento em lote? =

O recurso de processamento em lote escaneia sua biblioteca de mídia e processa todas as imagens que ainda não têm texto alternativo. Dependendo do número de imagens, este processo pode levar algum tempo para ser concluído.

= Este plugin funciona com sites multilíngues? =

Sim, você pode selecionar o idioma para o texto alternativo gerado para corresponder ao idioma do seu site. O plugin suporta vários idiomas, incluindo inglês, espanhol, francês, alemão, italiano, português, russo, chinês, japonês e hindi.
