Generera automatiskt SEO-optimerad alt-text, titlar och bildtexter för dina bilder med hjälp av AI-teknik. Förbättra tillgängligheten och öka sökrankingar.

== Beskrivning ==

**Automatisk Alt-text för Bilder** är ett kraftfullt WordPress-tillägg som använder toppmodern AI-teknik för att automatiskt generera högkvalitativ, SEO-optimerad alt-text för dina bilder. Genom att tillhandahålla beskrivande alt-text för alla dina bilder hjälper detta tillägg till att förbättra din webbplats tillgänglighet för synskadade användare och öka dina sökmotorrankingar.

### Huvudfunktioner

* **AI-driven analys**: Vår avancerade AI analyserar dina bilder för att generera korrekt, kontextuellt relevant alt-text.
* **SEO-optimering**: Alt-texten är optimerad för sökmotorer för att förbättra dina bilders rankning i sökresultat.
* **Tillgänglighetsefterlevnad**: Hjälper till att göra din webbplats mer tillgänglig och i enlighet med WCAG-riktlinjer.
* **Batchbearbetning**: Bearbeta alla befintliga bilder i ditt mediebibliotek med ett enda klick.
* **Automatisk bearbetning**: Generera automatiskt alt-text för nya bilder när de laddas upp.
* **Anpassningsbar utdata**: Välj att generera endast alt-text eller inkludera även titlar och bildtexter.
* **Flerspråkigt stöd**: Generera alt-text på flera språk för att stödja internationella målgrupper.
* **Användarvänligt gränssnitt**: Enkel, intuitiv inställningspanel för enkel konfiguration.

### Komma igång

1. Installera och aktivera tillägget.
2. Gå till **Inställningar > Automatisk Alt-text** för att konfigurera tillägget.
3. Ange din ForVoyez API-nyckel. (Har du ingen? [Skaffa en här](https://forvoyez.com))
4. Konfigurera dina inställningar för språk, utdataformat och andra alternativ.
5. Använd alternativet 'Batchbearbetning' för att generera alt-text för alla befintliga bilder.

### Tekniska detaljer

Tillägget bearbetar dina bilder säkert genom ForVoyez API för att generera korrekt och relevant alt-text. Dina bilder analyseras med hjälp av avancerade algoritmer för datorseende och språkbearbetning för att skapa beskrivningar som både är informativa för användare och optimerade för sökmotorer.

== Installation ==

1. Ladda upp tilläggsfilerna till katalogen `/wp-content/plugins/auto-alt-text-for-images`, eller installera tillägget direkt via WordPress-skärmen för tillägg.
2. Aktivera tillägget via 'Tillägg'-skärmen i WordPress.
3. Gå till **Inställningar > Automatisk Alt-text** för att konfigurera tilläggsinställningarna.
4. Ange din ForVoyez API-nyckel.
5. Konfigurera dina inställningar och spara dem.

== Vanliga frågor ==

= Varför behöver jag en API-nyckel? =

API-nyckeln krävs för att få tillgång till ForVoyez bildanalystjänster som driver genereringen av alt-text. Du kan få en API-nyckel genom att registrera dig på [https://forvoyez.com](https://forvoyez.com).

= Kan jag anpassa den genererade alt-texten? =

Ja, du kan konfigurera språk, maximal längd och optimeringsnivå för den genererade alt-texten. Dessutom kan du alltid manuellt redigera den genererade alt-texten efter att den har skapats.

= Vilka bildformat stöds? =

Tillägget stöder alla vanliga bildformat, inklusive JPEG, PNG, GIF, WebP och fler. Alla bildformat som WordPress stöder kan bearbetas av tillägget.

= Kommer detta tillägg att göra min webbplats långsammare? =

Nej, genereringen av alt-text sker i bakgrunden och påverkar inte din webbplats prestanda i frontend. När nya bilder laddas upp genereras alt-texten under uppladdningsprocessen utan att påverka användarupplevelsen.

= Hur fungerar batchbearbetningen? =

Batchbearbetningsfunktionen skannar ditt mediebibliotek och bearbetar alla bilder som inte redan har alt-text. Beroende på antalet bilder kan denna process ta lite tid att slutföra.

= Fungerar detta tillägg med flerspråkiga webbplatser? =

Ja, du kan välja språk för den genererade alt-texten för att matcha din webbplats språk. Tillägget stöder flera språk inklusive engelska, spanska, franska, tyska, italienska, portugisiska, ryska, kinesiska, japanska och hindi.
