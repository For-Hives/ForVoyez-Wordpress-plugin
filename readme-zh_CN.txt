== 描述 ==

ForVoyez 自动图像替代文本是一个强大的 WordPress 插件，它使用人工智能技术自动为您内容中的图像生成高质量、对搜索引擎友好的替代文本、标题和说明。这个插件可以轻松提高您网站的无障碍性和搜索引擎优化。

= 主要功能 =

* 🤖 AI 驱动的替代文本生成
* 🖼️ 批量处理现有图像
* 🔄 自动处理新上传的图像
* 🎨 可自定义的输出格式
* 🌐 多语言支持
* 🧰 用户友好的 WordPress 管理界面

我们的 AI 驱动的 API 使开发人员和内容创作者能够轻松优化图像以提高 SEO。使用我们强大的解决方案节省时间、提升 SEO 并简化您的工作流程。

== 安装 ==

1. 将插件文件上传到 `/wp-content/plugins/forvoyez-auto-alt-text-for-images` 目录，或直接通过 WordPress 插件界面安装插件。
2. 通过 WordPress 中的"插件"界面激活插件。
3. 使用设置 -> ForVoyez 自动替代文本界面来配置插件。
4. 在设置页面输入您的 ForVoyez API 密钥。

= 获取 API 密钥 =

1. 访问 [ForVoyez 注册页面](https://forvoyez.com/signup) 创建一个账户。
2. 登录后，导航到您的 [ForVoyez 仪表板](https://forvoyez.com/dashboard)。
3. 在仪表板的 API 部分生成一个 API 密钥。
4. 复制 API 密钥并粘贴到插件的设置页面。

== 常见问题 ==

= 我需要 API 密钥才能使用这个插件吗？ =

是的，您需要 ForVoyez API 密钥才能使用此插件。您可以通过在 [https://forvoyez.com](https://forvoyez.com) 注册来获取一个。

= 我可以自定义生成的替代文本吗？ =

虽然插件会自动生成替代文本，但如果需要，您始终可以手动编辑每个图像的生成文本。

= 这个插件适用于所有图像格式吗？ =

该插件适用于 WordPress 支持的大多数常见图像格式，包括 JPEG、PNG、GIF 和 WebP。

= 这个插件会减慢我的网站速度吗？ =

不会，插件在后台处理图像，不会影响您网站的前端性能。

= 批量处理功能如何工作？ =

您可以在媒体库中选择多个图像，并使用批量操作下拉菜单中的"生成替代文本"选项一次性处理所有图像。

= 生成的内容是多语言的吗？ =

AI 可以生成多种语言的内容。您可以在插件设置中指定您偏好的语言。
