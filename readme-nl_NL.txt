== Beschrijving ==

ForVoyez Automatische Alt-Tekst voor Afbeeldingen is een krachtige WordPress-plugin die AI-technologie gebruikt om automatisch hoogwaardige, SEO-vriendelijke alt-tekst, titels en bijschriften te genereren voor afbeeldingen in uw content. Deze plugin verbetert moeiteloos de toegankelijkheid en zoekmachineoptimalisatie van uw website.

== Installatie ==

1. Upload de plugin-bestanden naar de map `/wp-content/plugins/forvoyez-auto-alt-text-for-images`, of installeer de plugin rechtstreeks via het WordPress-pluginscherm.
2. Activeer de plugin via het 'Plugins' scherm in WordPress.
3. Gebruik het scherm Instellingen -> ForVoyez Automatische Alt-Tekst om de plugin te configureren.
4. Voer uw ForVoyez API-sleutel in op de instellingenpagina.

= Een API-sleutel verkrijgen =

1. Bezoek de [ForVoyez registratiepagina](https://forvoyez.com/signup) om een account aan te maken.
2. Eenmaal ingelogd, navigeer naar uw [ForVoyez dashboard](https://forvoyez.com/dashboard).
3. Genereer een API-sleutel in het API-gedeelte van uw dashboard.
4. Kopieer de API-sleutel en plak deze op de instellingenpagina van de plugin.

== Veelgestelde Vragen ==

= Heb ik een API-sleutel nodig om deze plugin te gebruiken? =

Ja, u heeft een ForVoyez API-sleutel nodig om deze plugin te gebruiken. U kunt er een krijgen door u te registreren op [https://forvoyez.com](https://forvoyez.com).

= Kan ik de gegenereerde alt-tekst aanpassen? =

Hoewel de plugin automatisch alt-tekst genereert, kunt u altijd handmatig de gegenereerde tekst voor elke afbeelding bewerken indien nodig.

= Werkt deze plugin met alle afbeeldingsformaten? =

De plugin werkt met de meeste gangbare afbeeldingsformaten die door WordPress worden ondersteund, waaronder JPEG, PNG, GIF en WebP.

= Zal deze plugin mijn website vertragen? =

Nee, de plugin verwerkt afbeeldingen op de achtergrond en heeft geen invloed op de frontend-prestaties van uw website.

= Hoe werkt de batchverwerkingsfunctie? =

U kunt meerdere afbeeldingen in uw mediabibliotheek selecteren en de optie "Genereer Alt-Tekst" gebruiken uit het dropdown-menu voor bulkacties om ze allemaal in Ã©Ã©n keer te verwerken.

= Is de gegenereerde inhoud meertalig? =

De AI kan inhoud in meerdere talen genereren. U kunt uw voorkeurstaal specificeren in de plugin-instellingen.
